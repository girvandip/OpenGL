# OpenGL 

---

## Prerequisites

1. CMake GUI
2. C++ Compiler
3. OpenGL

## Installation and Running

1. Cloen this repo
2. Open CMake GUI
3. Set `Where is the source code :` to this directory
4. Choose the suitable compiler
5. Set `Where to build the binaries :` to `./build` directory
6. Configure
7. Generate
8. Open `build` folder
9. Make and use the project
